TO RUN HANGMAN GAME:

1. Must be in a Windows or UNIX-type Environment
2. Must have java JRE 1.8 or JDK 1.8 installed
3. To run the game, open a console window and run the following command at the prompt:
java -jar myHangman.jar <language> <country-code>

For example:

# to run game in Canadian English
java -jar myHangman.jar en CA

# to run game in Canadian French
java -jar myHangman.jar fr CA

# to run game in US English 
java -jar myHangman.jar en US

# to run game in France French 
java -jar myHangman.jar fr FR
